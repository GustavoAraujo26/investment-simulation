Seja bem vindo ao simulador de investimentos!

Este é um site para realizar simulações de composição de uma carteira de investimentos. Nele é possível escolher vários ativos que estão disponíveis na B3, escolher o peso (porcentagem de participação) de cada ativo na sua carteira, e posteriormente, realizar a simulação de um aporte financeiro.

Sobre os ativos disponíveis, atualmente a aplicação conta somente com 3 tipos principais:

  • Ações
  • Fundos (podendo ser fundos imobiliários, FIAGROS, dentre outros disponíveis na B3)
  • BDR's

Sobre a fonte de dados do sistema, foi construída integração com a API da BRAPI. Além disso vale destacar que as cotações dos ativos disponibilizados são sempre DO DIA ÚTIL ANTERIOR (D - 1).
